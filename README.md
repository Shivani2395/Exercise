# Fetch Rewards

React App which fetches data from API and displays it on web page.

## Installation

Ensure you have Node installed on your device. Once installed run the commands written below. 

```bash
npm init
npm install
npm start
```

## To view Web App

```python
http://localhost:3000/
```
